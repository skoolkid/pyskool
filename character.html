character¶

Defines the Character class, which determines how characters are animated and how they behave.

class character.Character(character_id, name, head_xy=None, flags='')¶

Base class for anything in the game that moves.

Parameters:
  • character_id – Unique ID.
  • name – Display name.
  • head_xy – The coordinates of the character’s head within his sprite when he’s standing upright (used for collision detection).
  • flags – Character flags.

move()¶

Move the character. The steps taken are:

  1. make the character give lines (if he has a lines message buffered)
  2. check whether it’s time to move the character, and abort if not
  3. move the character from the midstride position (if he is midstride), or
  4. hand control of the character over to the current command in his command list
